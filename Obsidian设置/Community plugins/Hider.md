这插件用于隐藏 Obsidian 中的一些组件，可以使软件整体看上去更加简洁

插件设置比较简单：

![](https://img.aiexplorernote.com/pic/e-mail-forward20250921190519.png)

可以隐藏的组件是：

- **Hide tab bar (隐藏标签页栏)**
    
    - 描述：隐藏窗口顶部的标签页容器。
        
- **Hide status bar (隐藏状态栏)**
    
    - 描述：隐藏字数统计、字符数统计和反向链接计数。（**在您的截图中，此项为开启状态**）
        
- **Hide vault name (隐藏仓库名称)**
    
    - 描述：隐藏您的仓库名称。注意：这也会隐藏对“设置”和“仓库切换器”图标的访问权限，您需要使用热键或命令面板来打开它们。
        
- **Hide scroll bars (隐藏滚动条)**
    
    - 描述：隐藏所有的滚动条。
        
- **Hide sidebar toggle buttons (隐藏侧边栏切换按钮)**
    
    - 描述：隐藏左右两个侧边栏的展开/折叠按钮。（**在您的截图中，此项为开启状态**）
        
- **Hide tooltips (隐藏工具提示)**
    
    - 描述：隐藏所有的工具提示（鼠标悬停时出现的说明）。
        
- **Hide file explorer buttons (隐藏文件资源管理器按钮)**
    
    - 描述：隐藏文件资源管理器顶部的按钮（例如：新建文件、新建文件夹等）。
        
- **Hide instructions (隐藏提示信息)**
    
    - 描述：隐藏弹窗中的操作提示。
        
- **Hide search suggestions (隐藏搜索建议)**
    
    - 描述：隐藏搜索面板中的建议。
        
- **Hide count of search term matches (隐藏搜索词匹配计数)**
    
    - 描述：隐藏每个搜索结果中的匹配数量。
        
- **Hide properties in Reading view (隐藏阅读视图中的属性)**
    
    - 描述：在阅读视图中隐藏笔记的属性部分。
