- **`t:Script`**：查找所有 C# 脚本文件[](https://docs.unity3d.com/kr/6000.0/Manual/search-adb.html)。
    
- **`t:ComputeShader`**：用于 GPU 并行计算的 Compute Shader 资源。
    
- **`t:RenderTexture`**：运行时渲染目标纹理。
    
- **`t:Cubemap`**：用于天空盒或反射探针的立方体贴图。
    
- **`t:GameObject`**：查找场景中的游戏对象（在 Hierarchy 搜索时有效，而非 Project 窗口）
  
  - **组合搜索与排除**：
    
    - **多个条件**：`t:Texture2D t:Sprite` (查找所有 2 D 纹理和精灵)
        
    - **排除特定类型**：`-t:Texture2D` (在 Project 中输入此命令可过滤掉所有图片文件)[](https://docs.unity3d.com/cn/current/Manual/search-query-operators.html)
        
- **自定义类搜索**：
    
    - 如果你有一个继承自 `ScriptableObject` 或 `MonoBehaviour` 的脚本 `MyItemData`，你可以直接使用 **`t:MyItemData`** 来查找所有基于该脚本创建的配置文件[](https://docs-alpha.unity3d.com/cn/2018.4/ScriptReference/AssetDatabase.FindAssets.html)。
